# gc_website_2025

## Setup and Build

This project uses Pug for HTML templating.

### Prerequisites

- Node.js and npm installed.

### Installation

1.  Open a terminal in the project directory.
2.  Run `npm install` to install the required dependencies (Pug CLI).

### Building the Project

To generate the HTML files from the Pug templates:

```bash
npm run build
```

This command will compile all `.pug` files in the `views/` directory and output the corresponding `.html` files in the root directory.

### Development

To watch for changes and automatically rebuild:

```bash
npm run watch
```
